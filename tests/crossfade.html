imgCrossfade v0.1 / 04-02-2016

A jQuery Plugin to easily crossfade through images (same dimensions required)

Kevin KÃ¶llmann (Koelli91) - licensed under GPL-3.0


Usage

Include:

<script src="js/jquery.imgCrossfade.min.js"></script>

HTML:

<div id="cycler">
    <img src="img/one.jpg" />
    <img src="img/two.jpg" />
    ...
</div>

JS:

$(document).ready(function() {
    $("#cycler").imgCrossfade();
});

CSS:

#cycler {
    width: 227px; /* the common width of the images */
    max-width: 100%; /* for responsiveness - choose whatever max-width you like */
}

View in action:

Features:

  • Bootstrap-Compatible
  • Lightweight: ~1 KiB minified
  • Responsive
  • All browsers (incl. IE6)

Options:

$('#cycler').imgCrossfade({
    fading: 'fast'|'medium'|'slow'|<any number in ms>
    interval: <any number in ms>
});

/*
* fading: how long the crossfades should take (Default: 'medium' (= 1500 ms))
* interval: how often the image should change (Default: 7000 ms)
*/
